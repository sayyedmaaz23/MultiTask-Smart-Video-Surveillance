# MultiTask-Smart-Video-Surveillance
A real-time intelligent video surveillance system that combines ConvLSTM autoencoder-based anomaly detection, YOLO object detection, and ViViT transformer-based action recognition. Uses a multi-threaded pipeline to filter normal frames and classify suspicious video segments efficiently.
